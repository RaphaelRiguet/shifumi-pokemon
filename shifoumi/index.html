Le type plante gagne contre le type eau, le type eau gagne contre le type feu et le type feu gagne contre le type plante. Le jeu compte 10 parties au total et les points sont distribués de la façon suivante : 1 point pour le gagnant, 0 point pour le perdant à chaque partie, et 0 point en cas d'égalité. Au bout des 10 parties, le gagnant est affiché. Le joueur a également la possibilité de rejouer.
